# WorkWatch AI

Remote employee tracking system with face detection, activity monitoring, and automated payroll management.

## 📖 Documentation
A detailed guide to the project's architecture, features, and technical flow can be found in:
👉 **[PROJECT_DOCUMENTATION.md](./PROJECT_DOCUMENTATION.md)**

## 🚀 Getting Started
1. **Configure Environment**: Copy `.env.example` to `.env` and fill in your Supabase credentials.
2. **Setup Database**: Execute the SQL provided in `DATABASE_SETUP.md` in your Supabase SQL Editor.
3. **Install Dependencies**: `npm install`
4. **Development**: `npm run dev`

## 🛡️ Key Features
- **Biometric Attendance**: Face-verified start/stop functionality.
- **Admin Control**: Live activity monitoring and dashboard.
- **Automated Earnings**: Real-time wage calculation.
- **Team Communications**: Integrated real-time chat.
